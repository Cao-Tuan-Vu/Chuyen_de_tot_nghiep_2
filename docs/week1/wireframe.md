# Week 1 - Wireframe (text)

## Screen 1: Login/Register

- Email
- Password
- Toggle mode Login/Register
- Display name (chi hien khi Register)
- CTA: Dang nhap / Tao tai khoan

## Screen 2: Home

- Xin chao {displayName}
- Role hien tai
- Nut mo Profile
- Nut mo Khoa hoc
- Nut Dang xuat

## Screen 3: Profile

- Email (read-only)
- Role (read-only)
- Display name (editable)
- CTA: Luu thay doi

## Screen 4: Course List (Week 4)

- Danh sach khoa hoc
- title + level + mo ta ngan
- CTA: Mo khoa hoc

## Screen 5: Lesson List (Week 4)

- Danh sach bai hoc theo thu tu
- Chi tiet loai bai: Ly thuyet / Ly thuyet + Quiz
- CTA: Mo bai hoc

## Screen 6: Lesson Detail (Week 4)

- Tieu de bai hoc
- Noi dung ly thuyet
- CTA: Lam quiz bai nay (neu co)

## Screen 7: Quiz (Week 5)

- Danh sach cau hoi + cac lua chon
- Chon dap an cho tung cau
- CTA: Nop bai

## Screen 8: Quiz Result (Week 5)

- Tong diem (score/total)
- Danh sach cau dung/sai
- Giai thich ngan cho tung cau

## Screen flow

Login/Register -> Home -> Profile -> Home
Home -> Khoa hoc -> Danh sach bai hoc -> Chi tiet bai hoc -> Quiz -> Ket qua
Home -> Dang xuat -> Login
